# 鸡汤摘录

  1、我把生活看成一系列"角色扮演游戏"。我的任务就是以最适合我的"角色"的方式玩我的"游戏"。

游戏可以是任何东西，成为好学生、成为好父亲、做出明智的财务决策、领导一个项目、进行房屋翻新等等。

这样的好处是，我的压力减轻了很多，我开始以中立的眼光看待一切，不在意演员是谁（包括我自己），只是客观地思考怎样才能取得游戏的最佳结果。
