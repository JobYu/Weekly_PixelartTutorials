# 鸡汤摘录

1、利益总是自上而下分配，风险总是从下往上承担。

2、知识的诅咒（Curse of knowledge）是一种认知偏差，指的是与他人交流时，你不知不觉地假设对方拥有跟你相似的认知，能够理解你的意思。
