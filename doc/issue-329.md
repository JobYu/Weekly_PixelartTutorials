# 鸡汤摘录

1. - 如果AI可以随便出像素画，那这个像素画就没有商业价值。不是说这个像素画没价值，而是说这个像素画没有商业价值。
   - 就像空气，它虽然对人来说很重要，很有价值，但因为随便就能获得，那就没有商业价值。
   - 所以我们画像素画，要尽量画那些AI无法直接生成的像素画。
